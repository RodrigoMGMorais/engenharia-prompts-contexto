# 🧠 Engenharia de Prompts e Contexto

Repositório de estudo, laboratórios e prática sobre como estruturar instruções, gerenciar contexto e construir respostas confiáveis com Modelos de Linguagem Grande (LLMs).

---

## 🎯 Objetivo
Transformar o uso de Inteligência Artificial de "perguntas soltas" para **fluxos estruturados, repetíveis e com qualidade controlada**, aplicando técnicas usadas no mercado para reduzir alucinações, manter a precisão e otimizar recursos.
